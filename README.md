## Описание программы

Book Tracker - это GUI-приложение для ведения списка прочитанных книг. Приложение позволяет добавлять информацию о книгах (название, автор, жанр, количество страниц), фильтровать записи по жанру и количеству страниц, сохранять и загружать данные в формате JSON.

### Возможности:
- ✅ Добавление книг с указанием названия, автора, жанра и количества страниц
- ✅ Фильтрация записей по жанру (выбор из списка доступных жанров)
- ✅ Фильтрация по количеству страниц (больше заданного значения)
- ✅ Сохранение данных в JSON-файл
- ✅ Загрузка данных из JSON-файла
- ✅ Удаление выбранной книги из списка
- ✅ Автоматическое сохранение при добавлении/удалении
- ✅ Валидация вводимых данных (все поля обязательны, страницы - целое число)

## Инструкция по запуску

### Требования:
- Python 3.6 или выше
- Установленная библиотека Tkinter (обычно идёт в комплекте с Python)

### Запуск:
1. Откройте терминал или командную строку
2. Перейдите в директорию проекта:
   ```bash
   cd book_tracker
Запустите приложение:

bash
python book_tracker.py
Примеры использования
Добавление книги:
Заполните все поля формы:

Название книги: "Мастер и Маргарита"

Автор: "Михаил Булгаков"

Жанр: "Роман"

Количество страниц: "480"

Нажмите кнопку "Добавить книгу" — запись появится в таблице

Фильтрация:
По жанру:

Выберите жанр из выпадающего списка (например, "Роман")

Нажмите "Применить фильтр" — в таблице останутся только романы

По количеству страниц:

Введите число в поле "Страниц > " (например, "300")

Нажмите "Применить фильтр" — покажутся книги длиннее 300 страниц

Совместная фильтрация: фильтры по жанру и страницам работают одновременно

Сброс фильтров: нажмите "Сбросить фильтр" — покажутся все книги

Удаление книги:
Выберите книгу в таблице (кликните на строку)

Нажмите "Удалить выбранную книгу"

Книга исчезнет из списка

Сохранение и загрузка:
Сохранить в JSON — данные сохранятся в файл books.json

Загрузить из JSON — данные загрузятся из файла books.json

Примечание: При добавлении или удалении книги сохранение происходит автоматически.
